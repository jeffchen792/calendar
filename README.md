# Cosmic ✦ 雙星日曆

情侶的 3D 星系日曆 — 你們的時間是一個共同的世界。

## 技術棧
- Vite + React 19
- Tailwind CSS v3
- Zustand (狀態管理)
- React Three Fiber (3D — Phase 2)

## Phase 1 (當前)
- [x] 專案骨架
- [x] 雙人配對（邀請碼）
- [x] 事件 CRUD（你/他/我們）
- [ ] Supabase 即時同步
- [ ] 2D 月曆檢視

## Phase 2
